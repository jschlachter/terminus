## Overview

Join us for an informal, hands-on session exploring how AI coding assistants are changing the way we write, review, and ship software. Whether you're already using these tools daily or just curious about where to start, this session has something for everyone.

We'll cover real workflows — not just toy examples — and leave time for live Q&A and experimentation.

## What We'll Cover

### Inline Completions vs. Chat-Driven Development
We'll walk through the difference between accepting ghost-text completions and using a conversational interface to drive larger structural changes — and show when each approach shines.

### Test Generation in Practice
Watch a real test suite grow from zero coverage to ~80% using AI suggestions, then critique the output: catching hallucinated assertions, missing edge cases, and over-specified mocks.

### Refactoring Legacy Code
A live refactor of a gnarly utility function — renaming, extracting helpers, updating call sites — guided entirely through AI chat. We'll discuss how to prompt effectively and when to push back on suggestions.

### Security & Code Review
A quick look at how AI reviewers can catch common issues (SQL injection, missing input validation, insecure defaults) and where they fall short. Spoiler: they are not a substitute for a real security review.
