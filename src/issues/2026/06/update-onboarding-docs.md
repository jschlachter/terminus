## Issue Description

The onboarding documentation linked from the HR portal references:

- **JIRA** (replaced by Linear in Q1 2026)
- **IT Helpdesk Portal v1** (decommissioned 2026-03-15; the new portal is at `it.company.internal`)
- **Slack #general** setup steps (the channel was renamed to `#company-wide`)
- VPN setup instructions for GlobalProtect v5.2 (current version is 6.1; setup flow changed)

New starters have been raising IT tickets asking for help with steps that no longer match the documentation.

## Resolution

Documentation updated on 2026-07-02:

- Replaced JIRA references with Linear onboarding steps
- Updated IT helpdesk portal URL and login instructions
- Updated Slack channel names throughout
- Replaced GlobalProtect 5.2 screenshots with 6.1 equivalents

Updated document published to the internal wiki at `wiki.company.internal/onboarding`.

## Notes

Consider scheduling a quarterly documentation review to keep onboarding materials current with tooling changes.
